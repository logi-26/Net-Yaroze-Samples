# Memory Card Save and Load with Animated Icon
## CD-ROM Version
This sample demonstrates creating a save game file on the PlayStation memory card with a 3-frame animated save game icon.

## How It Works

The project loads 3 16x16 TIM files through the CD-ROM:

```txt
\\DATA\\ICON1.TIM
\\DATA\\ICON2.TIM
\\DATA\\ICON3.TIM
